Данное мобильное приложение предназначено для сканирования и создания Qr-кодов под нужды пользователя. APK файл находится по пути: app/release/app-release.apk. Работает с Android 8.0 Oreo и выше.
Для создания Qr-кодов в приложении требуется зайти во вкладку "Генератор" и написать текст, что будет отображён на Qr-коде, а для сканирования различных Qr-кодов требуется зайти во вкладку "Сканер",
нажать кнопку "Сканировать", предоставить доступ к камере и навестись на нужный Qr-код.
